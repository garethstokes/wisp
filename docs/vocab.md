# Vocabulary

Activity: Canonical record of an item (email, calendar event, feedback)
WorkItem: Unit of agent execution
ToolCall / ToolResult: Structured side‑effect interface
Receipt: Immutable audit record
Confidence: Float 0–1 measuring certainty
Autonomy Tier: Levels 1–4 governing agent authority
Surface: Any user‑visible message
Option framing: Present choices, never instructions
